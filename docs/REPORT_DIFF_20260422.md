# 2802 レポート差分サンプル (修正前 vs 修正後)

## 修正前（問題あり）

- ギャップ予測: `+5.00%`（N225欠損時でも固定化）
- α成分: `NaN` なのに `100/100` 扱い
- シナリオ: 窓埋め確率 `0%` でも窓埋めTPを採用
- レジーム: `RISK_ON + BEARISH` を未解釈で並列表示
- 重要イベント: 4/22停戦期限情報が欠落

## 修正後（本対応）

- ギャップ予測:
  - `N225` 欠損時は `UNAVAILABLE / gap_pct=None`
  - `CME_N225` で自動フォールバック可能
- α成分:
  - `compute_alpha_score(NaN) -> 50.0`
  - レポート上は `α: N/A (計算不能・中立扱い)`
- G0シナリオ:
  - 窓埋め確率に応じてTPを25%/50%へ縮小
  - `RRR<1.0` は `EXCLUDED`
- レジーム:
  - `RISK_ON + BEARISH => BULL_FADING`
  - サイズ係数を自動縮小
- イベント:
  - `米イラン停戦期限切れ` を当日CRITICAL表示
  - `action_bias=REDUCE_POSITION` 時にサイズ係数を追加圧縮
