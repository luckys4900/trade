【📋 はじめに】
────────────────────────────────────────────────────────────────────────────────

このシステムは、大口トレーダー（クジラ）の取引を監視し、彼らのシグナルに基づいて
自動的にBTC取引を実行する高度なコピートレーディングシステムです。

3つの独立した並行システムが24時間稼働します：
  1. Whale Monitor（クジラ監視）
  2. Macro Filter（市場環境判定）
  3. Main Trading Bot（自動売買実行）


【🚀 クイックスタート - 5分で始める】
────────────────────────────────────────────────────────────────────────────────

ステップ1: システムを起動
  01_START.lnk をダブルクリック

ステップ2: リアルタイムダッシュボード表示
  02_DASHBOARD.lnk をダブルクリック（自動更新で稼働状況を監視）

ステップ3: システムを停止（終了時）
  04_STOP.lnk をダブルクリック


【🎯 各ファイルの役割 - わかりやすく】
════════════════════════════════════════════════════════════════════════════════

■ 01_START.lnk / 01_Start_Whale_System.bat
  役割: システム起動
  用途: 毎日のシステム開始時に1回クリック
  動作: 3つのシステムを自動起動
  結果: バックグラウンドで稼働開始

■ 02_DASHBOARD.lnk / 02_DASHBOARD.bat ⭐ 最も重要！
  役割: リアルタイムダッシュボード（CMDで監視）
  用途: 毎日の稼働状況確認
  機能:
    • 5秒ごと自動更新
    • プロセス状態表示
    • シグナル（direction, strength）表示
    • 市場環境（regime, ATR）表示
    • ログの最新行表示
  終了: Ctrl+C を押す

■ 03_STATUS.lnk / 03_Check_Status.bat
  役割: 詳細ステータス確認（1回限り）
  用途: JSON形式で詳細データを見たいとき

■ 04_STOP.lnk / 04_Stop_System.bat
  役割: システム停止
  用途: 取引を終了するとき

■ 05_VIEW_LOGS.lnk / 05_VIEW_LOGS.bat
  役割: ログビューア（詳細ログ確認）
  用途: エラー診断、システム詳細動作確認

■ 00_GUIDE.lnk / 00_GUIDE.bat
  役割: このガイドを表示
  用途: 使い方を忘れたときに確認


【📁 自動生成ファイル（自動更新される）】
════════════════════════════════════════════════════════════════════════════════

■ whale_signal.json (15分ごと更新)
  内容: クジラシグナル（LONG/SHORT/NONE）
  見方:
    "valid": true  = シグナル有効（取引チャンス到来！）
    "valid": false = シグナルなし（様子見）
    "direction": "LONG" = 買い信号
    "direction": "SHORT" = 売り信号
    "strength": 0～1.5 = シグナルの強さ

■ macro_state.json (60分ごと更新)
  内容: 市場環境（ボラティリティ、経済指標）
  見方:
    "regime": "NORMAL" = 通常モード（取引実行可）
    "regime": "EXTREME" = 経済指標周辺（取引スキップ）
    "caution_mode": true = ボラティリティ高い

■ logs/ ディレクトリ
  内容: 3つのシステムの詳細ログ
  ファイル:
    • whale_monitor_*.log (クジラ監視)
    • macro_filter_*.log (マクロフィルタ)
    • qwen_unified_live.log (メインボット)

【⚠️  FAQ - よくある質問】
════════════════════════════════════════════════════════════════════════════════

Q: valid=false が続いているけど大丈夫？
A: はい、正常です。「取引チャンスがない」という意味です。

Q: CMDウィンドウが表示されない
A: システムはバックグラウンド実行です。02_DASHBOARD.bat で確認してください。

Q: whale_signal.json が更新されない
A: 初回は最大15分待ってください。それでも更新されなければ05_VIEW_LOGS.batでエラーを確認。

Q: プロセスが勝手に停止する
A: 05_VIEW_LOGS.bat でログを見てエラー内容を確認してください。

Q: システムを再起動したい
A: 04_STOP → 01_START でOKです。

Q: ログファイルはどこ？
A: logs/ フォルダ内（05_VIEW_LOGS.bat でも確認可能）
