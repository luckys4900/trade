【1分で理解する】
════════════════════════════════════════════════════════════════════════════════

このシステムは 3つのプログラムが同時に動作します：
  1. whale_monitor.py   → 大口トレーダーを監視（15分ごと）
  2. macro_filter.py    → 市場環境をチェック（60分ごと）
  3. qwen_unified_live.py → 自動売買実行（1分ごと）


【毎日の操作 - たった2つのファイル】
════════════════════════════════════════════════════════════════════════════════

朝:
  ① 01_START.lnk をダブルクリック → システム起動完了

日中:
  ② 02_DASHBOARD.lnk をダブルクリック → 稼働状況確認
     （5秒ごと自動更新で監視）

夜:
  ③ 04_STOP.lnk をダブルクリック → システム停止完了

【システムの出力 - 何が起こるか】
════════════════════════════════════════════════════════════════════════════════

自動生成ファイル（勝手に作られ、勝手に更新される）:

1️⃣  whale_signal.json （15分ごと更新）
    役割: クジラシグナル
    重要な項目:
      "valid": true  → ✓ シグナル有効（取引チャンス！）
      "valid": false → ○ シグナルなし（様子見）
      "direction": "LONG"  → 買いシグナル
      "direction": "SHORT" → 売りシグナル

2️⃣  macro_state.json （60分ごと更新）
    役割: 市場環境フィルター
    重要な項目:
      "regime": "NORMAL"   → 通常モード（取引実行可）
      "regime": "EXTREME"  → 経済指標周辺（取引スキップ）
      "caution_mode": false → 安全
      "caution_mode": true  → ボラティリティ高い（注意）

3️⃣  logs/ フォルダ
    役割: 詳細ログ記録
    ファイル:
      whale_monitor_*.log   → クジラ監視ログ
      macro_filter_*.log    → マクロフィルタログ
      qwen_unified_live.log → 自動売買ログ

【トラブルシューティング - よくある問題】
════════════════════════════════════════════════════════════════════════════════

問題1: 02_DASHBOARD.lnk でプロセスが 3個 以下
  原因: システムが停止している
  対処:
    1. 04_STOP.lnk で停止
    2. 01_START.lnk で再起動

問題2: "valid=false" が続く
  原因: シグナルがない（取引チャンスなし）
  対処: 待機中です、正常動作です

問題3: ERROR メッセージが表示される
  原因: システムエラー
  対処:
    1. 05_VIEW_LOGS.lnk でログ確認
    2. エラー内容をメモ
    3. 04_STOP → 01_START で再起動

問題4: whale_signal.json が更新されない
  原因: Whale Monitor が動いていない
  対処:
    1. 02_DASHBOARD.lnk で確認
    2. 04_STOP → 01_START で再起動
    3. 最大15分待つ（初回）
