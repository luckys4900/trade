================================================================================
PROBLEM ROOT CAUSE
================================================================================

Issue: UTF-8 BOM encoding in batch files
  - Batch files created with UTF-8 (with BOM)
  - CMD interprets BOM as visible characters: EF BB BF
  - These appear as: 絶武笊 (garbled output)
  - Subsequent commands fail to parse

Solution: ASCII-only encoding
  - Rewritten all batch files as pure ASCII
  - No special characters (only ASCII printable)
  - Direct calls to Python for complex operations
  - Clean execution with no corruption
