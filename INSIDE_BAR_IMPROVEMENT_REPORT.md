## 💡 **実装上の注意点**

### 1. Trailing Stop の実装詳細
```python
highest_price = max(highest_price, current_price)
stop_loss = highest_price * (1 - trail_pct/100)
if current_price <= stop_loss:
    EXIT()
```

### 2. リスク管理
- Position Size: 基本 MAX_POS_PCT = 40%
- Risk: 2% per trade
- Max Losses: 5連敗でクールダウン (3bar)
- Drawdown Halt: 25%に達したら全ポジション決済

### 3. 推奨的な段階的実装
```
Phase 1: 紙取引で 1-2 週間検証
Phase 2: 最小サイズ (0.01 BTC等) でライブ
Phase 3: サイズ拡大 (0.05-0.1 BTC)
Phase 4: フル実装
```
