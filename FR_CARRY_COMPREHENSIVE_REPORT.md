| 指標 | 値 | 判定 |
|------|-----|------|
| **平均資金調達料** | 0.0096% | ⚠️ 低い |

### 根本原因

```
月間EV = (資金調達料 × 回数/月) - (手数料 × 回数) - (価格逆行損失)
      = (0.0096% × 45回) - (0.17% × 45回) - (0.48% × 45回)
      = 0.43% - 7.65% - 21.6%
      = -28.8% ← 赤字
```

**主要問題:**
1. 資金調達料 (0.0096%) が手数料 (0.17%) に遠く及ばない
2. ホールド期間中の価格逆行リスク (-0.48%)
3. サンプルサイズが小さく統計的信頼性が低い (n=230)

#### 結論
- **推奨**: 8h保有 (相対的に被害が少ない)
- **理由**: 保有期間が短いほど価格逆行リスクが低い

**詳細な期待値式:**
```

8h:  0.0096 × 1 - 0.17 - 0.2286 = -0.3891% ✓最小損失
24h: 0.0096 × 1 - 0.17 - 0.4827 = -0.6432% 
48h: 0.0096 × 1 - 0.17 - 0.4497 = -0.6101%

※ FRは8時間ごと1回だけカウント (計算方法の確認が必要)
```

## 付録: 詳細計算

### 期待値式の完全形

```
月間EV = Σ[(FR × ホールド日数) - (手数料 × 2) - (相場逆行損失)]

具体例 (FR > 0.015%, 24h保有, Maker exit, 複合EX):
  FR: 0.015%
  取引数/月: 1.5回
  期待値/trade: +0.44%
  
  月間EV = 0.44% × 1.5回 = 0.66%
  月間$ = 0.66% × $190 = $1.25/月
  年間$ = $1.25 × 12 = $15/年
```

### 統計的有意性の計算

```python
import scipy.stats as stats

# Current data
n = 230
ev = -0.006432  # -0.64%
se = 0.2857778 / np.sqrt(230)  # Standard error

# t-test vs zero
t_stat = ev / se
p_value = 2 * (1 - stats.t.cdf(abs(t_stat), n-1))

# p_value = 0.25 (not significant)
# Conclusion: Cannot reject H0 that true EV = 0
```
