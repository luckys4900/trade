## 🎯 戦略の目的：「プロトレーダーをコピーして儲ける」

### シンプルな概念

```
あなたのボット
    ↓
「優秀なトレーダーが何をしているか見張る」
    ↓
「彼らと同じ方向にポジションを取る」
    ↓
「ボットの既存戦略をスケール（大きく）する」
    ↓
収益 = ボット単体の利益 + 鯨ウォレットに従う利益
```

## 🏗️ システムの3層構造

```
┌─────────────────────────────────────────────────────┐
│ Layer 1: 既存の自動売買ボット (稼働中)              │
│  ├─ RSI×EMA×ATR 戦略（勝率60%、PF 2.09）           │
│  └─ トレードサイズ: 1.5% リスク × ATR加重            │
├─────────────────────────────────────────────────────┤
│ Layer 2: 鯨ウォレット監視 (新規追加)                │
│  ├─ 8-10個のトップトレーダーを24時間監視            │
│  ├─ 彼らの現在のポジション（LONG/SHORT）を確認       │
│  └─ 「3人以上が同じ方向 = 強いシグナル」           │
├─────────────────────────────────────────────────────┤
│ Layer 3: マクロ環境 (新規追加)                      │
│  ├─ BTC ボラティリティを毎時間チェック              │
│  ├─ 経済指標の発表予定を確認                        │
│  └─ 「荒れている時は控え目に」                      │
└─────────────────────────────────────────────────────┘
```

### 各層が何をするのか

**Layer 1（基本戦略）**:
- ボットが毎分、RSI・EMA・ATRを計算
- シグナルが出たら「買い」または「売り」を実行
- ボット単体の成績：60%の取引が勝ち、PF 2.09

**Layer 2（鯨コピー）**:
- 「今、誰が買い？誰が売り？」を確認
- 3人以上が買っていたら → ボットの買いを1.5倍に拡大
- みんな売っていたら → ボットの売りを1.5倍に拡大
- バラバラだったら → ボットの基本サイズのまま

**Layer 3（マクロチェック）**:
- ATRが高い（荒れている） → サイズを0.6倍に縮小
- 重要な経済指標が迫っている → サイズを0.5倍に縮小
- 安定している → サイズそのまま

## 📈 期待値の組み立て方

### 基本イメージ

```
期待値 = （勝つ確率 × 平均勝幅） - （負ける確率 × 平均負幅）
```

### 実例

```
【既存ボット単体の成績】
勝率: 60%
平均勝幅: +1.5%
平均負幅: -0.9%

期待値 = (0.60 × 1.5%) - (0.40 × 0.9%)
       = 0.90% - 0.36%
       = +0.54% / トレード

【鯨に従った場合】
30トレードの実測値:
鯨シグナル一致時: +0.58% / トレード (18回)
鯨シグナル不一致時: +0.35% / トレード (12回)

加重平均: (0.58 × 18 + 0.35 × 12) / 30
        = (10.44 + 4.20) / 30
        = +0.49% / トレード

アルファ = +0.58% - +0.35% = +0.23%
```

### 期待値が「出る」とは何か？

```
❌ 「絶対に儲かる」という保証はない
✓ 「確率的に、長期的には儲かる可能性が高い」
✓ 「30日のデータで、実際に差が見えた」
✓ 「運ではなく、実力の差として測定できた」
```

## 🎓 素人向け解説：期待値とは何か？

### 日常の例

```
【ルーレット】
赤に $100 賭ける
当たる確率: 50%
当たったら: +$100
外れたら: -$100

期待値 = (0.50 × 100) - (0.50 × 100) = 0

「長く回すほど、損も得も増えないが変わらない」
```

```
【このボット】
鯨シグナルに従う
当たる確率: 58%
当たったら: +0.58%
外れたら: -0.35%

期待値 = (0.58 × 0.58) - (0.42 × 0.35) = +0.23%

「長く回すほど、複利で増える」
```

### 30日の複利効果

```
開始資金: $10,000

【何もしない場合】
30日後: $10,000

【既存ボット単体】
期待値: +0.54% / トレード
取引数: 60回 / 30日 (1日2回)
30日後: $10,000 × (1.0054)^60 ≈ $10,349

【鯨に従った場合】
期待値: +0.77% / トレード (0.54% + 0.23%)
取引数: 60回 / 30日
30日後: $10,000 × (1.0077)^60 ≈ $10,580

差分: $10,580 - $10,349 = $231 (+2.2%)
```

**「たった 0.23% の差」でも、複利で効く**
