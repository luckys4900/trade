# マルチエージェントAIトレードシステム - 実装完了

## 概要
RTKプロキシを導入し、複数のAIエージェントを連携させる高性能自動売買システムを構築しました。月額 $2,000 のサブスクリプションをオープンソースで代替し、トークン使用量を60-90%削減します。

## 構成要素

### 1. アーキテクチャ
```
Master Agent (Ollama Qwen3 8b) → Multi-Agent Orchestration
├── Frontend Agent (GLM-5.1) → ユーザーインターフェース管理
├── Backend Agent (GLM-5-Turbo) → API/データ処理  
├── Analysis Agent (GLM-5.1) → 戦略分析と最適化
├── Research Agent (GLM-4.7-Flash) → マクロ調査
├── Protection Agent (GLM-5-Turbo) → リスク管理
└── Local Fallback (Qwen3 8b) → トークン切れ時のフォールバック
```

### 2. コアファイル
- `multi_agent_system.py` - 統合システム
- `master_orchestrator.py` - マスター・オーケストレーター
- `agents.py` - 各エージェントの実装
- `setup_rtk_proxy.py` - RTKプロキシのセットアップ
- `dashboard.html` - Webダッシュボード
- `multi_agent_config.json` - 設定ファイル

### 3. RTKプロキシ統合
- トークン使用量を60-90%削減
- クラウド/ローカルモデルの自動切り替え
- キャッシュとバッチ処理の最適化

## インストール手順

### 1. 依存ライブラリのインストール
```bash
pip install aiohttp redis numpy python-dotenv
```

### 2. RTKプロキシのセットアップ
```bash
python setup_rtk_proxy.py
```

### 3. システムの起動
```bash
python multi_agent_system.py
```

またはWindowsバッチファイルで:
```bash
start_multi_agent.bat
```
