## パラメータ設定

```python
# short_trading_bot.py の config

'rsi_overbought': 70,           # RSI過買い水準
'profit_target_pct': 0.005,     # 利益確定 (0.5%)
'stop_loss_pct': 0.01,          # 損切り (1%)
'max_hold_bars': 10,            # 最大保有時間 (10時間)
'check_interval': 60,           # チェック間隔 (60秒)
```

### パラメータ調整ガイド

| パラメータ | 調整内容 | 効果 |
|-----------|---------|------|
| profit_target_pct | 減らす (0.3%) | 決済が早い、トレード増加 |
| profit_target_pct | 増やす (1.0%) | 利益確定が遅い、トレード減少 |
| stop_loss_pct | 減らす (0.5%) | リスク制限きつい、損失少 |
| stop_loss_pct | 増やす (2.0%) | リスク許容大、勝率改善 |
| max_hold_bars | 減らす (5) | 決済早い、リスク制限 |
| max_hold_bars | 増やす (20) | ポジション長期保有 |

## リスク管理

### 構成されたリスク制限

✅ **1トレード最大損失**: stop_loss_pct で自動制限
✅ **ポジション保有時間**: max_hold_bars で強制決済
✅ **利益確定**: profit_target_pct で自動実行

### 推奨される追加対策

1. **日次最大損失制限**: -5% を超えたらボット停止
2. **レバレッジ制限**: 1倍（Hyperliquid デフォルト）
3. **ポジションサイズ制限**: 口座資金の10%以下

### ログ読み方

- **SHORT ENTRY SIGNAL**: ショートを開始したタイミング
- **PROFIT TARGET HIT**: 利益確定
- **STOP LOSS HIT**: 損切り実行
- **TIMEOUT**: 時間切れで決済
