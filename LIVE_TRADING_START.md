## ⚡ リアルタイム対応

### Q: エントリーが出ない

**A: RSI < 60 なら正常です**

```
RSI < 60 = シグナルなし
→ エントリーが出ないのは正常な動作
```

### Q: スリップページが大きい

**A: ボラティリティが高い時間帯の可能性**

```
対応:
1. ログでスリップページ%を確認
2. 0.1% 以上なら check_interval を 120秒に延長
3. より流動性の高い時間帯を待つ
```

### Q: 連続で負けが出た

**A: 2-3負けは統計的に正常です**

```
勝率98%でも:
100トレード中2回は負け
1000トレード中20回は負け

連続3負け以上なら相場環境の確認が必要
```

