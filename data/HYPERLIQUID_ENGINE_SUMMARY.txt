KEY FEATURES
================================================================================

Kelly Criterion Position Sizing:
  Formula: fractional_kelly = (bp + p - q) / b × 0.55
  Where p=win_rate, q=1-p, b=loss_ratio
  Result: Conservative position sizing (1-25% per trade)

Multi-Level Risk Management:
  1. Daily Loss Limit (-5%)
  2. Position Size Limit (25% of capital max)
  3. Position Stop-Loss (-2.5%)
  4. Trailing Stop (0.75%)
  5. Emergency Stop (-10% drawdown)

Position Lifecycle:
  Entry → Monitoring → Stop-Loss/Trailing Stop → Exit
  - Tracks entry/exit prices and times
  - Calculates unrealized P&L continuously
  - Records peak profit and peak loss
  - Maintains complete position history

Integration with Clarity Act Core:
  - Uses signals from clarity_act_core.py
  - Applies dynamic parameters from timeline manager
  - Supports BTC/ETH pair trading strategy
  - Compatible with backtest parameters (60% win rate, +1% expected return)

USAGE EXAMPLE - COMPLETE TRADING FLOW
================================================================================

from hyperliquid_executor import HyperliquidExecutor
from position_manager import PositionManager
from risk_manager import RiskManager
from capital_manager import CapitalManager

# Initialize
executor = HyperliquidExecutor(
    api_key="YOUR_KEY",
    api_secret="YOUR_SECRET",
    paper_trade=True,  # Start with paper trading
    kelly_fraction=0.55
)
position_mgr = PositionManager()
risk_mgr = RiskManager(initial_capital=10000)
capital_mgr = CapitalManager(initial_capital=10000)

# Calculate position size
position_size = executor.calculate_position_size(
    account_balance=10000,
    expected_return=1.0,      # 1% from backtest
    win_rate=0.60,            # 60% win rate
    stop_loss_percent=-2.5
)

# Validate entry
valid, msg = risk_mgr.validate_entry('BTC/USDC', position_size)
if valid:
    # Allocate capital
    capital_mgr.allocate_capital(position_size, purpose="position")

    # Execute trade
    result = executor.execute_market_order(
        symbol='BTC/USDC',
        side='buy',
        position_size_usd=position_size,
        current_price=50000
    )

    # Track position
    position_mgr.open_position(
        symbol='BTC/USDC',
        side='buy',
        entry_price=50000,
        quantity=position_size / 50000,
        position_size_usd=position_size
    )

# During trading - continuous monitoring
while trading_active:
    # Update prices
    position_mgr.update_position_price('BTC/USDC', current_price)

    # Monitor risk
    metrics = risk_mgr.monitor_position_risk(
        symbol='BTC/USDC',
        entry_price=50000,
        current_price=current_price,
        position_size_usd=position_size,
        side='buy'
    )

    # Check for stops
    if metrics['stop_loss_triggered']:
        pnl, pnl_percent = position_mgr.close_position(
            'BTC/USDC',
            exit_price=current_price,
            exit_reason="Stop loss"
        )
        capital_mgr.release_capital(position_size, reason="position_closed")
        break
