# BTC/USDT 4H ADAPTIVE RSI v5 - LIVE TRADER

Hyperliquid用の自動取引ボットで、RSI（相対力指数）戦略を使用してBTC/USDTを取引します。

## 特徴

- **適応型RSI戦略**: 買われすぎ/売られすぎのレベルに基づいて自動取引
- **4時間足チャート**: 中期トレンドに焦点を当てた取引
- **リスク管理**: ポジションサイズの制限と最大ポジション数の設定
- **自動ログ記録**: すべての取引アクティビティを記録
- **バックテスト機能**: 過去のデータで戦略をテスト可能

## セットアップ方法

1. **APIキーの設定**:
   - HyperliquidからAPIキーとシークレットキーを取得
   - `config.json` ファイルを編集してキーを入力

2. **必須ライブラリのインストール**:
   - `start_trader.bat` を実行すると自動でインストールされます

3. **パラメータ設定** (オプション):
   - `config.json` で取引パラメータを調整可能

## 使用方法

### ライブトレードの開始
- `start_trader.bat` をダブルクリックして取引ボットを起動

### バックテストの実行
- `run_backtest.bat` をダブルクリックしてバックテストを実行
- 過去180日間のデータで戦略の性能をテスト

### 停止
- `stop_trader_all.bat` をダブルクリックしてすべてのPythonプロセスを停止
- または起動したコンソールで Ctrl+C を押して停止

## 設定パラメータ

- `api_key`: Hyperliquid APIキー
- `api_secret`: Hyperliquid APIシークレット
- `symbol`: 取引通貨 (デフォルト: BTC)
- `timeframe`: 時間足 (デフォルト: 4h)
- `rsi_period`: RSI期間 (デフォルト: 14)
- `rsi_overbought`: 買われすぎレベル (デフォルト: 70)
- `rsi_oversold`: 売られすぎレベル (デフォルト: 30)
- `position_size_usd`: ポジションサイズUSD (デフォルト: 100)
- `max_positions`: 最大ポジション数 (デフォルト: 1)
- `leverage`: レバレッジ (デフォルト: 10)
- `check_interval`: チェック間隔（秒） (デフォルト: 60)

## 取引ロジック

1. **買いシグナル**: RSIが売られすぎレベル（30）以下
2. **売りシグナル**: RSIが買われすぎレベル（70）以上
3. **ポジション管理**: 最大ポジション数を超えないように管理

## バックテスト

バックテスト機能を使用して、戦略の過去のパフォーマンスを確認できます：

- **期間**: デフォルトで過去180日間
- **初期資金**: $10,000
- **スリッページ**: 0.05%
- **手数料**: 0.02%
- **出力**: 
  - コンソールにパフォーマンス指標を表示
  - 詳細なレポートをJSONファイルで保存
  - 資産推移チャートをPNGファイルで保存

### バックテスト結果の指標

- **総取引数**: 実行された取引の総数
- **勝率**: 利益を上げた取引の割合
- **損益**: 総利益/損失（USDとパーセント）
- **最大ドローダウン**: 最大資産減少率
- **シャープレシオ**: リスク調整後の収益率
- **最終残高**: バックテスト終了時の残高

## 注意事項

- 本番口座で使用する前に、十分なバックテストを行ってください
- バックテスト結果は過去のパフォーマンスを示すものであり、将来の結果を保証するものではありません
- 取引にはリスクが伴います。自己責任でお願いします
- 初期設定では小額でテストすることをお勧めします

## ログ

- すべての取引アクティビティは `trader_YYYYMMDD_HHMMSS.log` に記録されます
- バックテストレポートは `backtest_report_YYYYMMDD_HHMMSS.json` に保存されます
- エラー発生時のデバッグに役立ちます

## サポート

問題が発生した場合は、ログファイルを確認してください。
