## ⚙️ パラメータ説明

### デフォルト設定（推奨）

```python
rsi_overbought = 70        # RSI過買い水準
profit_target_pct = 0.005  # 利益確定 0.5%
stop_loss_pct = 0.01       # 損切り 1.0%
max_hold_bars = 10         # 最大保有 10時間
```

### パラメータ調整ガイド

**より多くのトレード を実行したい場合:**
```python
profit_target_pct = 0.003  # 0.3% に低下
max_hold_bars = 5          # 5時間に短縮
```

**より堅実な運用にしたい場合:**
```python
profit_target_pct = 0.01   # 1.0% に上昇
stop_loss_pct = 0.005      # 0.5% に厳格化
```

## ❓ よくある質問

### Q1: 現在のBTC相場でいつまで使える？

**A**: ダウントレンドが続く限り有効です。RSI > 70のシグナルが出続ける限り機能します。

相場環境が「レンジ」に変わったら、グリッド戦略への切り替えも検討できます。

### Q2: 手数料はちゃんと考慮されている？

**A**: はい。メーカー手数料 0.015% × 往復（0.03%）を引いた純利益で計算されています。

実際の利益はこの計算結果から若干少なくなる可能性があります（スリップページなど）。

### Q3: Hyperliquid API取得方法は？

**A**: Hyperliquid公式サイトから:
1. メタマスク接続
2. API キー生成
3. Secret Key をコピー
4. config.json に貼り付け

### Q4: ペーパートレードでどのくらい実行すべき？

**A**: 最低 **1週間以上** をお勧めします。

- 1週間で約 10-15 トレード
- 勝率とドローダウンを確認
- パラメータ調整が必要か判断

### Q5: ライブトレード開始時の資金は？

**A**: 推奨: **$1,000～$5,000** から開始

- 1トレード平均 $948 の利益が可能
- リスク: 最大 -0.56% = 約 $56
- 十分な利益化を確認してから増額
