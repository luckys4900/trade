## エグゼクティブサマリー

### 問題の特定
FR Z-Score Mean Reversion戦略は以下の致命的な問題を抱えていました：

```
パフォーマンス：
  IN-SAMPLE (2024-04-12 ～ 2025-03-31): +0.24% EV (n=678 trades)
  OUT-OF-SAMPLE (2025-04-01 ～ 2026-03-02): -0.16% EV (n=574 trades)
  統計有意性: p = 0.000042 (有意だが**負の収益**を確認)
```

### 根本原因分析（3層）

#### 1. **レジーム変化（市場構造の劇的変化）**

| 指標 | IS | OOS | 変化 |
|------|----|----|------|
| 平均FR | +0.1801 bps | +0.1105 bps | -38.6% ↓ |
| FR ボラティリティ | 0.2059 bps | 0.1291 bps | -37.3% ↓ |
| 統計検定 | - | - | **p < 0.000001** ✗ |

**解釈**: OOS期間でFRレベルが全体的に低下し、ボラティリティも減少。

#### 2. **メジャーリバージョン効果の喪失**

最重要な発見：

```
Long シグナル (z < -3σ) の6時間後の期待リターン：
  IS: +0.3395% (n=155, std=1.40%)
  OOS: +0.0020% (n=228, std=1.18%)
  劇的な99.4%の効果低下 ↓↓↓

Short シグナル (z > 3σ) の6時間後の期待リターン：
  IS: +0.1379% (n=243, std=1.16%)
  OOS: -0.0010% (n=115, std=1.06%)
  逆転して負になっている ↓
```

**根本原因**: 2025年4月以降、FRのMean Reversion効果が完全に消滅

#### 3. **取引コスト > 平均利益**

```
エリアバー = ATR(14)の5倍で目標利益を狙う場合：

IS期間：
  平均ATR: 574.58 USDT
  粗利益ターゲット (5x ATR): +3.778%
  取引コスト (往復): -0.170%
  ネット利益: +3.608% ✓ (余裕あり)

OOS期間：
  平均ATR: 584.47 USDT
  粗利益ターゲット (5x ATR): +2.926%
  取引コスト (往復): -0.170%
  ネット利益: +2.756% ✓ (危険水準)
  実効EV: -0.160% ✗ (負)

原因: MR効果の喪失により、取引コストを取り返せない
```

## 改善案とその効果

### 改善1: TP目標の引き上げ (5x → 8x ATR)

**対策**:
- TP距離を5倍 → 8倍ATRに拡大
- 粗利益ターゲット: +2.9% → +4.7%
- 取引コスト: -0.17% (変わらず)
- ネット利益: +2.76% → +4.53%

**効果**: 
- 取引コストを確実にカバー
- MR効果の低下に耐性がつく

**トレードオフ**:
- ホールド時間が長くなる (6時間 → 10時間)
- ウィンレート低下の可能性

### 改善2: 高信頼度フィルター (RSI + MACD)

**対策**:
```
エントリー条件:
1. Z-score閾値: 3.5σ (FRが極端に乖離)
2. RSI確認:
   - LONG (z < -3.5): RSI < 30 必須
   - SHORT (z > 3.5): RSI > 70 必須
3. MACD確認:
   - LONG: MACD ヒストグラム < 0
   - SHORT: MACD ヒストグラム > 0
```

**効果**:
- 信号数: 265 → 123 (46%削減)
- **信号質の改善**: 弱いMR信号をフィルタリング
- 統計有意性の向上

### 改善3: レジーム検出 (Mean Reversion Strength)

**対策**:
```
各取引時点で過去120時間を分析：
- 直近の z < -2.5 シグナル後の6時間リターン を測定
- 直近の z > 2.5 シグナル後の6時間リターン を測定
- 平均MR効果 > 50 bps の場合のみエントリー
- MR効果 <= 50 bps ならスキップ (レジーム外)
```

**効果**:
- 市場環境に応じた適応的なトレード
- 明らかに機能しない時期を避ける

## 重要な発見と結論

### 1. **FR Z-Score Mean Reversion は 2024年4月には有効だったが、2025年4月に完全に破綻した**

- 可能性1: 市場参加者の学習による無裁定化
- 可能性2: 長期傾向の変化 (FRのボラティリティ低下)
- 可能性3: 高周波取引業者による吸収

### 2. **統計有意性 (p < 0.05) と戦略の価値は独立**

- v4の結果: OOS EV -0.16%, p = 0.000042
- 「統計的に有意」=「負の収益が有意に確認された」という意味
- 統計有意性だけで戦略を導入することは危険

### 3. **パラメータ最適化はIS/OOS乖離を拡大する**

- σ 2.5 (広い) → IS: +0.38%, OOS: +0.08% (乖離小)
- σ 3.5 (狭い) → IS: +0.71%, OOS: -0.15% (乖離大)
- **結論**: パラメータの調整では問題解決不可


## 統計検定の詳細

### p-value の解釈

```
帰無仮説 H₀: EV = 0 (戦略は有効ではない)
対立仮説 H₁: EV ≠ 0 (戦略は有効)

v4 最良結果 (EMA_2.5):
  OOS EV: -0.15957%
  n = 574 trades
  t-statistic: -4.127
  p-value: 0.000042

解釈:
  p < 0.05 → 帰無仮説を棄却
  つまり: 「EV = 0 ではない」ことを95%の信頼度で確認
  しかし: 実際の EV は負 → 統計的に有意な**損失戦略**
```

### 必要なサンプル数

p < 0.05 を達成するには：

```
現在の平均リターン: -0.16%
標準偏差: 2.08%
有意義な差: 0.1% (10 bps)

必要サンプル数 = (1.96 * σ / Δ)² 
             = (1.96 * 0.0208 / 0.001)²
             ≈ 16,800 trades

現在進捗: 574 / 16,800 = 3.4%
```

もし OOS を続けても、意味のある正の結果を得るには**数年必要**。
