このフォルダには、毎日実行するファイルが入っています。

================================================================================
【毎日の実行手順】
================================================================================

STEP 1: MASTER_LAUNCHER.bat をダブルクリック
         ↓
         メニューが表示されます

STEP 2: メニューから選択

  [1] START SYSTEM
      朝、システムを起動します
      所要時間: 5秒
      
  [2] MONITOR (推奨)
      リアルタイムで状態確認します
      所要時間: 5分
      複数回実行（日中3-4回）
      
  [3] STOP SYSTEM
      夜、システムを停止します
      所要時間: 5秒
      
  [4] LOGS
      トラブル時にログを確認
      
  [5] GUIDE
      使い方のヘルプ

STEP 3: 完了したら enter キーを押す

================================================================================
【毎日の実行パターン】
================================================================================

朝:
  1. MASTER_LAUNCHER.bat をダブルクリック
  2. [1] START SYSTEM を選択
  3. Enter キーを押す
  4. 完了

日中 (3-4回):
  1. MASTER_LAUNCHER.bat をダブルクリック
  2. [2] MONITOR を選択
  3. 画面で確認（更新は5秒ごと）
  4. Ctrl+C で終了
  
夜:
  1. MASTER_LAUNCHER.bat をダブルクリック
  2. [3] STOP SYSTEM を選択
  3. Enter キーを押す
  4. 完了

================================================================================
【わからないことがあったら】
================================================================================

  → MASTER_LAUNCHER.bat を起動
  → [5] GUIDE を選択
  → 画面で説明を読む

  または

  → ../DOCS/ フォルダを開く
  → ドキュメントを読む

================================================================================
【その他のフォルダ】
================================================================================

../DOCS/       ← ドキュメント・ガイド（読むだけ）
../SYSTEM/     ← システムコア（触らない）
../DATA/       ← 出力ファイル（自動生成）
../ARCHIVE/    ← 古いファイル（削除可能）
